## ✨ Overview

**Webraft Co.** is a studio portfolio and landing page built to showcase web design and development services for small businesses across India. The site features a sharp, modern SaaS aesthetic inspired by Notion, Linear, and Raycast — with clean borders, high whitespace, and a professional blue/zinc color palette.

---

## 🚀 Features

- **Responsive Design** — Mobile-first layout with hamburger nav, adaptive grids, and fluid typography
- **Smooth Animations** — Framer Motion powered scroll reveals, hover effects, and page transitions
- **Live Contact Form** — EmailJS integration with validation, honeypot protection, and success animation
- **Book a Call** — Calendly modal embed for instant lead scheduling
- **WhatsApp Float** — One-tap WhatsApp CTA for instant client communication
- **Community Comments** — Supabase-powered real-time comments visible to all visitors
- **Demo Projects** — Standalone HTML showcase sites (Restaurant, Healthcare, E-commerce)
- **Accessibility** — Skip-to-content, aria-labels, keyboard navigation support
- **Local SEO Ready** — Structured for Kakinada, Andhra Pradesh targeting

## ⚙️ Getting Started

### Prerequisites
- Node.js 18+
- npm or yarn

### Installation

```bash
# Clone the repository
git clone https://github.com/webraftco-rgb/webraft-portfolio.git

# Navigate into the project
cd webraft-portfolio

# Install dependencies
npm install
```

### Environment Variables

Create a `.env` file in the root directory:

```env
VITE_EMAILJS_SERVICE_ID=your_service_id
VITE_EMAILJS_TEMPLATE_ID=your_template_id
VITE_EMAILJS_PUBLIC_KEY=your_public_key
VITE_SUPABASE_URL=your_supabase_url
VITE_SUPABASE_ANON_KEY=your_supabase_anon_key
```

### Run Locally

```bash
npm run dev
```

### Build for Production

```bash
npm run build
```
